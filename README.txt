1.	Install GO programming to your device.	Download the timeserver 
	compressed file and save timeserver.go to the directory of Go.2.	Run the command terminal.3. 	In the terminal, go to the directory of where the timeserver.go is 
	located.4. 	In the command terminal, type in go run timeserver.go 	4a.	Go to the default port http://localhost:8080/time to view the 
		current time	4b. 	If you go to http://localhost:8080 it should display a message 
		that you have the wrong URL page	4c. 	Click control c to terminate the program5. 	In the command terminal, type in go run timeserver.go ÑV 
	to view the version of the program6.	In the command terminal, type in go run timeserver.go Ñport 9090	to view the web server in port 9090. Replace 9090 with a different 
	port number of your choosing. Repeat steps 4a-4c and replace the 
	default port number with the port number that you chose to view 
	the web server page.ÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑÑ
